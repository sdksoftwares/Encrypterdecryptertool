Help & Support

We're here to assist you with any issues or questions you may have while using XCrypto. Follow the sections below for guidance or contact us directly if needed.


Frequently Asked Questions (FAQs):

1. How do I encrypt a file?

Select a file from the file dropdown menu or add a directory using the "Add Directory" button.
Enter a password in the password field.
Click the "Encrypt" button. Thus the temporary .enc file is saved at the default directory do not use the save as prompt appears after "temporary file save" it is to save your file info in.txt format

2. How do I decrypt a file?

Select an encrypted file (with .enc extension) from the file dropdown menu.
Enter the correct password used during encryption.
Click the "Decrypt" button to retrieve the original file.

3. How can I hide data in an image?

Use the "Hide in Image" button to select an image file.
Enter the text or data you want to hide and choose a save location for the encoded image.

4. How do I extract hidden data from an image?

Click the "Reveal Image" button, select the image file, and retrieve the hidden data.

5. Can I verify the integrity of my files?

Yes, use the "File Verify" option to check file integrity with SHA256, MD5 hashes, or file size verification methods.



Common Issues and Troubleshooting:

1. I forgot the encryption password. What can I do?
- Unfortunately, for security reasons, XCrypto cannot recover forgotten passwords. Ensure you store your password securely in the Key Library.

2. My file doesn't decrypt properly.
- Ensure you are using the same password and file format. A mismatch will cause errors.

3. The app crashes during a task.
- Make sure the file size is within acceptable limits, and check if your system has enough resources to process the task.

4. I can't load an image for steganography.
- Ensure the image format is supported (.png, .jpg, .bmp, etc.).
